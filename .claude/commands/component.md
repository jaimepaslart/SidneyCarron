Crée un nouveau composant Astro.

Demande :
1. Le nom du composant (PascalCase)
2. Une description courte de son rôle

Puis crée le fichier `.astro` dans `src/components/` avec :
- Les props typées en TypeScript
- Le markup HTML sémantique
- Les styles scopés avec Tailwind
- Un commentaire en tête décrivant le composant

Si le composant est lié à une section spécifique (galerie, header, footer...), propose de le mettre dans un sous-dossier approprié.
